Konfiguracja ProFTPd

Najważniejsze Ustawienia



Zmiany nazwy serwera

ServerName "Twój Server"

Czas po którym Połączenie FTP się wyłącza

TimeoutNoTransfer 600

Maksymalny czas połączenia bez wysyłania lub pobierania plików.


TimeoutStalled 600

Ogólna długość połączenia bez robienia czegokolwiek.


TimeoutIdle 1200

Konfiguruje maksymalną liczbę sekund, przez które proftpd pozwoli klientom pozostać połączonymi bez otrzymania jakichkolwiek danych. Jeśli dane zostaną odebrane, licznik czasu bezczynności jest resetowany. Ustawienie TimeoutIdle na 0 całkowicie wyłącza TimeoutIdle


Wiadomość powitalna podczas logowania

DisplayLogin welcome.msg

Ścieżka pliku którego treść zostanie wyświetlona podczas pierwszego logowania. Nazwa pliku może być względna lub bezwzględna. W przypadku nazwy względnej, plik jest szukany w początkowym katalogu, do którego użytkownik trafia zaraz po zalogowaniu (katalog domowy dla logowań użytkowników unixowych, katalog anonymous-root dla logowań anonimowych).


Wiadomość która pojawi się zawsze gdy zmienimy katalog

DisplayChdir .message true

Tekstowa nazwa która będzie wyświetlana za każdym razem gdy zmieni się katalog.


Jak szczegółowe ma być polecenie ls

Dokumentacja ListOptions "-l" 
    -1 Wyświetla leden plik na linię

    -A List all files except "." and ".."
      
    -a Wyświetla wszystkie ukryte pliki
      
    -C Wyświetla za pomocą kolumn
      
    -d List directory entries instead of directory contents
      
    -F Dodaje wskaźnik typu pliku (jeden z "*", "/", "=", "@" lub "|") do nazw plików
      
    -h Wyświetla rozmiary plików w czytelnym formacie(np. 1K, 234M, 2G)
      
    -L List files pointed to by symlinks
      
    -l Użyj długiego formatu listowania
      
    -n Wyświetlanie numerycznych identyfikatorów UID/GID zamiast nazw user/group
      
    -R Lista podkatalogów rekursywnie
      
    -r Sortuj nazwy plików w odwrotnej kolejności
      
    -S Sortuj po nazwie pliku
      
    -t Sortuj po czasie ostatniej modyfikacji
